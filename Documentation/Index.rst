Bootstrap3 responsive images
=============================================================

.. only:: html

	:Classification:
		lef_responsive_images

	:Version:
		|release|

	:Language:
		en

	:Description:
		Extends the Bootstrap Package (bootstrap_package)

		- Auto generate images with the right size for every breakpoint / layout
		- Support for retina
		- Exteded support of javascript based lazy-load, <picture> and <img srcset> methods

		   Responsive design is great, and providing the right image size on any device is a challenge.
